# Shidoka Charts

A data visualization library of framework-agnostic web components built with Chart.js.

## Getting Started

View the [Readme here](https://github.com/kyndryl-design-system/charts).

## Chart.js

These charts are driven by [Chart.js](https://www.chartjs.org). Please see the [documentation](https://www.chartjs.org/docs/latest/) for more fine-grained control and [examples](https://www.chartjs.org/docs/latest/samples/information.html).

## Foundation

This library is built on top of the [Shidoka Foundation](https://github.com/kyndryl-design-system/foundation) which has [it's own separate Storybook documentation](https://kyndryl-design-system.github.io/foundation/).
