# Color Palettes

All chart types have a default Shidoka color palette. You can choose an alternative by passing one of the below codes to an attribute `colorPalette` in the chart component `options` prop.

NOTE: The following color palettes are deprecated and will be removed in the next major version.(`default`, `rainforest`,`gentletouch`,`savannahsunrise`,`crystalcavern`,`magenta`,`magentanight`,`braziliansunrise`,`tequilasunrise`,`eveninghorizon`,`energizing`,`cherryblossom`,`springgreen`)
